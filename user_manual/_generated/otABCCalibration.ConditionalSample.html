class ConditionalSample(sample, intervalCriteria, inputColumnsId, distributionInput=None, outputColumnsId=None)¶

Parameters:
sampleclass Sample

the sample to be conditionned

intervalCriteriaclass Interval

Interval value to filter the sample, interval dimension has to be equal to sample one

inputcolumnsIdsequence of int

Sample column id to be considered as input

distributionInputclass Distribution, optionnal

distribution associated to inputSample (identified by its column input id), default will estimate the input distribution with kernelSmoothing

outputColumnssequence of int, optionnal

Sample columns id used to build outputSamle, defaut will consider no outputs
