FuncResidualFunctionOptimalPoint¶

class FuncResidualFunctionOptimalPoint(model, observedParameterIndices, toCalibrateParameterIndices, observedOutputIndices, inputObservations, outputObservations)¶

A class to compute residuals between model predictions and observed outputs for ABC calibration.

This class extends OpenTURNS Python function to compute residuals between model predictions and observed outputs for Approximate Bayesian Computation (ABC) calibration purposes. It takes observed parameters and parameters to calibrate as input and returns residuals.

Parameters:
modelFunction

The model to be calibrated

observedParameterIndiceslist of int

Indices of observed parameters in the model

toCalibrateParameterIndiceslist of int

Indices of parameters to calibrate in the model

observedOutputIndiceslist of int

Indices of observed outputs in the model

inputObservationsSample

Sample of observed input data

outputObservationsSample

Sample of observed output data

Examples

>>> # Create a simple model for testing
>>> model = ot.SymbolicFunction(['x1', 'x2', 'x3'], ['x1*x2 + x3'])
>>>
>>> # Define observed and calibration parameters indices
>>> obs_param_indices = [0, 1]
>>> calib_param_indices = [2]
>>> obs_output_indices = [0]
>>>
>>> # Create input and output observations
>>> input_obs = ot.Sample([[1.0, 2.0], [1.5, 2.5]])
>>> output_obs = ot.Sample([[3.0], [4.0]])
>>>
>>> # Create the residual function
>>> residual_func = FuncResidualFunctionOptimalPoint(
>>>     model, obs_param_indices, calib_param_indices, obs_output_indices,
>>>     input_obs, output_obs
>>> )
